- 메모를 테이블에 붙일 수 있게 한다. (드래그 앤 드롭)
  - 테이블에 붙이면 테이블 상단에 작은 아이콘으로 표시
  - 아이콘에 마우스 호버 시 메모 내용이 표시
  - 테이블 위치를 이동해도 메모는 그 자리에 붙어다님.
  - 드래그로 빈 곳에 놓으면 일반 메모로 돌아간다.
