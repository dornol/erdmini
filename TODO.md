# erdmini 기능 개선 목록

## Quick Wins (소규모, 높은 효과)

### 1. 키보드 단축키 도움말 패널
- `?` 키 또는 도움말 버튼으로 단축키 목록 모달 표시
- Ctrl+Z/Shift+Z, Ctrl+A, Ctrl+D, Space+drag, 방향키 팬 등 숨겨진 기능 노출
- 난이도: 소

### 2. 잠긴 테이블 캔버스 표시
- TableCard에 잠금 아이콘/배지 표시
- 현재는 TableEditor에서만 잠금 상태 확인 가능
- 난이도: 소

### 3. 사이드바 그룹 접기 상태 저장
- localStorage에 접힌 그룹 목록 저장, 새로고침 시 복원
- 난이도: 소

### 4. FK 참조 무결성 경고
- NOT NULL 컬럼에 ON DELETE SET NULL 설정 시 경고
- 잘못된 조합 방지
- 난이도: 소

### 5. 사이드바 검색 확장
- FK 관계, 인덱스 유무, 잠금 상태 등으로 필터링
- 현재는 테이블명/컬럼명/코멘트만 검색 가능
- 난이도: 소~중

### 6. 컬럼 복사/복제
- 같은 테이블 또는 다른 테이블로 컬럼 설정 복사
- audit_created_at 같은 반복 컬럼을 빠르게 생성
- 난이도: 소

### 7. 컬럼 기본값 프리셋
- 타입별 자주 쓰는 기본값 드롭다운 (NOW(), CURRENT_TIMESTAMP, UUID(), 0 등)
- 난이도: 소

---

## 중간 규모 개선

### 8. 테이블 일괄 편집
- 다중 선택된 테이블의 color, group, comment를 한번에 수정하는 다이얼로그
- 난이도: 중

### 9. 스키마 검증/린팅
- PK 누락, 순환 FK, 중복 인덱스, NOT NULL FK에 SET NULL 등 안티패턴 자동 체크
- 경고 목록 패널 표시
- 난이도: 중

### 10. PDF 내보내기
- 현재 SVG/PNG/Mermaid/PlantUML 지원, PDF 추가
- 인쇄/공유에 더 보편적인 포맷
- 난이도: 중

### 11. DDL 내보내기 포맷 옵션
- 들여쓰기 스타일, 따옴표 방식, 코멘트 포함 여부 등 커스텀 설정
- 난이도: 중

### 12. 실행취소 히스토리 패널
- 스크롤 가능한 히스토리 목록, 특정 시점으로 점프
- 현재는 Ctrl+Z/Shift+Z로만 탐색 가능
- 난이도: 중

### 13. 컬럼 숨기기 토글
- 캔버스에서 특정 컬럼 숨기기 (PK/FK만 표시 모드 등)
- 큰 테이블이 많을 때 가독성 향상
- 난이도: 중

### 14. 스키마 버전 비교 (Diff)
- 두 스키마 버전 비교, 변경사항 하이라이트
- undo 히스토리 또는 업로드된 파일 간 비교
- 난이도: 중

## 대규모 개선

### 15. FK 라인 스마트 라우팅
- FK 선 겹침 방지, 스플라인 경로 최적화
- 현재 단순 베지어 곡선으로 많은 FK 시 가독성 저하
- 난이도: 대

### 16. 사이드바 대규모 스키마 가상 스크롤
- 1000+ 테이블 시 DOM 렌더링 최적화
- 난이도: 중

### 17. 접근성 (A11y) 개선
- ARIA 레이블, 키보드 네비게이션, 스크린 리더 지원
- 난이도: 대

- 자연어 설명으로 스키마 자동 생성 ("블로그 시스템 만들어줘")
- LLM 연동 필요
- 난이도: 대

### 19. 실시간 DB 연결 & 역공학
- 라이브 데이터베이스 접속해서 스키마 자동 추출
- 현재는 DDL 텍스트 임포트만 가능
- 난이도: 대

### 20. 스키마 스냅샷/브랜칭
- 이름 붙인 스냅샷 생성, 스키마 변형 실험 후 전환 (git branch처럼)
- 난이도: 대
